Project Overview

Restaurant Finder is a React-powered food discovery app that helps users explore restaurants, browse menus, and view ratings. Inspired by platforms like Swiggy and Zomato, it provides a smooth and engaging interface for food lovers. The app is built using React, Redux, and Tailwind CSS, ensuring a modern and responsive user experience.

Users can search for restaurants by name, cuisine, or location, view detailed menu items, and even add their favorite dishes to the cart. The Redux-powered state management ensures a fast and seamless performance, making food discovery effortless.
